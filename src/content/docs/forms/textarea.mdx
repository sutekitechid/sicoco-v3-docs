A `<textarea>` is an HTML element for inputting multi-line text in web forms. It's typically used for comments or longer messages.

## Component Usage
### Default

A default `<textarea>` is a form element that users can freely interact with. It allows text input, editing, and scrolling. It is fully functional unless modified with attributes like disabled or readonly.

<STextarea client:only="vue" class="mt-4" />
<Code code={STextareaRaw} lang="vue" title="textarea.vue" />

### Disabled

A disabled `<textarea>` is a form element that users can see but not interact with or edit. It is rendered with a gray background and is non-clickable.

<STextareaDisabled client:only="vue" class="mt-4" />
<Code code={STextareaDisabledRaw} lang="vue" title="textareaDisabled.vue" />

### Readonly

A readonly `<textarea>` allows users to view but not edit the content. Unlike a disabled textarea, it can still be selected and copied. It typically has a normal appearance, indicating that it is editable but not in this case.

<STextareaReadonly client:only="vue" class="mt-4" />
<Code code={STextareaReadonlyRaw} lang="vue" title="textareaReadonly.vue" />

### Required

To make the `<textarea>` required, you can add the required attribute to the textarea element and wrap it with the `s-form-input` component.
