Sicoco is a private package, you need to login to npm with your GitHub account to install it. To do this, you need to generate a personal access token from your GitHub account.
<Steps>
1. Go to Settings -> Developer Settings -> Personal access tokens (skip if you already have one).
2. Run `npm login -registry=https://npm.pkg.github.com -scope=@sutekitechid` when you switch projects/repositories.
3. Enter your username and password. Use the token you generated as the password.
4. Run the following command to install Sicoco:
</Steps>

```bash
$ npm install @sutekitechid/flowbite-vue
```
